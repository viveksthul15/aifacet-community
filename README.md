# ◆ AIFacet Community

Community home for **[aifacet.dev](https://aifacet.dev)** — practical tricks, guides, news, and tools for developers who build with AI.

## What happens here

- **💬 Comments** — the comment box on every aifacet.dev post stores its threads in this repo's [Discussions](../../discussions)
- **❓ Q&A** — stuck on something from a trick or guide? [Ask here](../../discussions/categories/q-a)
- **💡 Ideas** — suggest a topic, trick, or tool you want covered
- **🐛 Tool bugs** — issues for AIFacet's free Windows tools are tracked on each tool's own repo (linked from its page on the site)

## House rules

Be specific, be kind, no spam. Questions with commands and error messages get answered fastest.
